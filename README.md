# ML_coursework2

Steps to run:
1. install the requirements - you may need to change the torch version to be suitable with your cuda install
2. download the data,checkpoints, and embeddings_external and move them into the root branch from this link below:
https://huggingface.co/KishPrak/simclr_checkpoint/tree/main
3. run active_learn.py, inference will take a few minutes but it'll produce results in terminal and a graph showing how the accuracy changes as budget changes
